## 概述

### 浅色主题

保持现有温暖配色，变量名统一为上述命名体系。

### 切换逻辑

```
页面加载 → 读取 localStorage('theme')
  ├── 有值 ('dark'|'light') → 应用
  └── 无值 → 检测 matchMedia('prefers-color-scheme: dark')
            ├── 匹配 → dark
            └── 不匹配 → light

用户手动切换 → 写入 localStorage + 立即切换 <html data-theme>
系统偏好变化 → 仅在没有手动设置时跟随（监听 change 事件）
```

### 切换 UI

- 设置面板底部增加「主题」行：🌙 深色 / ☀️ 浅色 两个图标按钮
- 切换时添加 CSS transition 平滑过渡（300ms ease）

## 二、新功能

### 2.1 深色/浅色模式切换

| 项目 | 说明 |
|------|------|
| 入口 | 设置面板底部 |
| 存储 | `localStorage('theme')`: `'dark'` \| `'light'` |
| 跟随系统 | 未手动设置时监听 `matchMedia` change 事件 |
| 过渡 | CSS `transition` 在 `body` 上应用 300ms 平滑过渡 |

### 2.2 历史统计图表

**数据存储：**
```js
dailyHistory: {
  "2026-05-31": { pomodoros: 8, focusMinutes: 200 },
  "2026-05-30": { pomodoros: 6, focusMinutes: 150 },
  // ... 最多保留 90 天
}
```

**图表类型：**
- **周视图（默认）：** 7 天柱状图
- **月视图：** 30 天柱状图，标注每日番茄数

**绘制方式：** 纯内联 SVG（零依赖），响应式宽度

**交互：**
- 统计面板下方新增可折叠的图表区域
- 周/月切换按钮
- 悬停柱状条显示具体日期和数量
- 今日柱子高亮

**SVG 图表组件规格：**
- 画布：320×180 viewBox 或响应式
- 柱状条：圆角矩形，填充色为 `--accent`
- 底部日期标签（周视图：周一到周日，月视图：每 5 天标注一次）
- 顶部数值标签（≥4 个番茄的柱子显示数字）
- Y 轴网格线（虚线，低透明度）

## 三、代码架构优化

### 3.1 计时器精度修复

**问题：** `setInterval(fn, 1000)` 存在累积漂移，长时间运行误差可达数秒。

**方案：** 基于 `Date.now()` 的目标时间纠偏法：
```js
function timerTick() {
  const elapsed = Math.floor((Date.now() - tickStart) / 1000);
  timeLeft = Math.max(0, totalTime - elapsed);
  updateDisplay();
  if (timeLeft <= 0) onTimerComplete();
}
```

**页面可见性处理：**
- 监听 `visibilitychange`：页面隐藏时记录时间戳，恢复时重新计算时间差
- 避免页面在后台时计时器「冻结」导致不准确

### 3.2 渲染优化

- **缓存所有 DOM 引用**：启动时一次性获取，存入 `dom` 对象
- **按需更新**：计时器每秒仅更新 3 个节点（时间文本、SVG 偏移、页面标题）
- **统计面板惰性更新**：仅在番茄完成或设置变更时刷新
- **使用 `requestAnimationFrame`** 防抖 SVG 属性更新

### 3.3 存储优化

- **防抖写入**：300ms debounce，避免频繁写入 localStorage
- **数据版本号**：`version: 2`，便于未来数据结构迁移
- **历史裁剪**：dailyHistory 最多保留 90 天，超出自动删除旧数据
- **结构化存储 key**：统一使用 `pomodoro_v2` 作为 key，避免与旧数据冲突

### 3.4 代码分区

单文件内重新组织为 11 个清晰分区：

```
1. CSS 变量 + 全局样式
2. HTML 结构
3. JS: 常量定义 + DOM 缓存
4. JS: 状态管理 + 存储函数
5. JS: 计时器核心逻辑
6. JS: 渲染函数（显示更新）
7. JS: 任务管理
8. JS: 设置管理 + 主题切换
9. JS: SVG 图表绘制
10. JS: 事件绑定
11. JS: 初始化
```

## 四、兼容性与边界

| 场景 | 处理方式 |
|------|----------|
| 首次访问 | 跟随系统主题，默认 25 分钟专注 |
| 旧数据迁移 | 检测 `pomodoro_data`，自动读取并写入新格式 |
| localStorage 满 | try/catch 包裹，静默失败 |
| 页面后台运行 | visibilitychange 校准计时器 |
| 90 天历史裁剪 | 每次保存时检查并删除过期数据 |
| SVG 图表无数据 | 显示「暂无数据，开始你的第一个番茄吧」占位文字 |
